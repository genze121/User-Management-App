
<h1>USERS MANAGEMENT APP SNAPSHOTS</h1>


Users Management Application Developed using JSP, Servlet, JDBC and MySQL Database.

## Following features
- Add Users
- Edit Users
- List Users
- Delete Users
- Login Model
- Registration Model
- Responsive layout

## Technology stack
- Java 8+
- JSP, HTML5, CSS3, BOOTSTRAP4 and Servlet
- JDBC
- MySQL database
- Eclipse IDE
- Tomcat server 8+
